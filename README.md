# Душоново: сайт-помощник по обращениям из-за отключений электричества

Это статический сайт для жителей села Душоново. Он помогает подготовить тексты обращений в ведомства, но не отправляет обращения за людей.

## Что внутри

- `index.html` — весь интерфейс и логика сайта.
- `config.js` — ссылки, ID полей Яндекс.Формы, ведомства и ручная статистика.
- `outages.js` — канонический журнал сильных эпизодов отключений.
- `.nojekyll` — служебный файл для GitHub Pages.

## Важные ограничения

- Не храните персональные данные жителей в репозитории.
- Сайт не просит пароли и не заходит в Госуслуги.
- Персональные данные хранятся только в `localStorage` браузера пользователя до перехода в Яндекс.Форму.
- Полный текст обращения не передаётся в URL Яндекс.Формы.
- Статистика считается только по подтверждённым отправкам: номер обращения, скрин, автоответ или файл.

## Как поменять Яндекс.Форму

Откройте `config.js` и замените:

```js
formUrl: "https://forms.yandex.ru/u/.../"
```

Затем замените ID вопросов в `formFields`.

Для вопросов типа «Один вариант» нужны не только ID вопроса, но и ID вариантов ответа. Они лежат в `formAnswerIds`.

## Как обновить журнал отключений

Замените файл `outages.js` новым каноническим файлом. Не кладите в репозиторий сырой экспорт Telegram-чата и персональные данные.

## Как обновить ручную статистику

В `config.js` поменяйте блок:

```js
stats: {
  lastUpdated: "2026-05-09",
  participants: 0,
  confirmedAppeals: 0,
  receivedAnswers: 0
}
```

## Как включить GitHub Pages

1. Откройте репозиторий на GitHub.
2. Перейдите в `Settings`.
3. Слева выберите `Pages`.
4. В `Build and deployment` выберите `Deploy from a branch`.
5. Branch: `main`.
6. Folder: `/root`.
7. Нажмите `Save`.
8. Через несколько минут сайт появится по адресу вида:
   `https://vasiedd.github.io/dushonovo-light/`

## Как тестировать перед запуском

Проверьте с телефона:

1. Открывается главная страница.
2. Без ФИО, телефона, почты, адреса и галочки сайт не формирует обращения.
3. Данные сохраняются после закрытия вкладки.
4. Кнопка «Очистить данные» удаляет данные.
5. Текст обращения копируется.
6. Кнопка ведомства открывает официальный сайт.
7. Кнопка «Я отправил» открывает Яндекс.Форму с подставленными данными.
8. Кнопка «Получил ответ» открывает Яндекс.Форму с подставленными данными.
9. Журнал отключений отображается.

## Как запускать кампанию

Сначала дайте ссылку 3–5 людям и проверьте, где они спотыкаются. Массово публикуйте только после теста.


## Как поменять ссылку «Сказать спасибо»

В `config.js` сейчас указана ссылка:

```js
thanksUrl: "https://ozon.ru/s/zharsous",
```

Если понадобится, замените её на прямую ссылку на карточку товара, витрину или поиск по магазину на Ozon.

Если хотите временно отключить переход, оставьте пустую строку:

```js
thanksUrl: "",
```

Кнопка «Сказать спасибо» находится внизу сайта и не влияет на работу обращений.
